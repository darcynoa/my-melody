# 🎵 My Melody

![enter image description here](https://i.makeagif.com/media/9-08-2015/DdPNzC.gif)

## **Personal Music Sovereignty for the Digital Age.**

Inspired by the precision of Eric B. & Rakim and built for those who want to own their library. **My Melody** is a local-first downloader and player pipeline that bridges the gap between YouTube Music and your personal device.

### 🌃 The Vibe

- **Nocturnal Aesthetic:** Deep shadows, neon lilac accents, and high-contrast typography.
- **Sovereignty:** Your music stays on your hardware. No subscriptions, no tracking, just the files.
- **Spirit Resonance:** UI transitions driven by GSAP to match the flow of the music.

### 🛠️ Built With

This project stands on the shoulders of giants:

- **[FFmpeg](https://ffmpeg.org/):** The architect. Handles the heavy lifting of audio conversion and metadata injection.
- **[yt-dlp](https://github.com/yt-dlp/yt-dlp):** The engine. A powerful command-line media downloader.
- **[Next.js](https://nextjs.org/):** The skeleton. Providing a modern, performant React framework for the UI.
- **[GSAP](https://greensock.com/gsap/):** The pulse. Powering fluid, high-fidelity animations
- **[Tailwind CSS](https://tailwindcss.com/):** The skin. Rapidly styling the gritty, neo-noir interface.

### 🚀 Getting Started

#### 1. Prerequisites

Ensure you have the following installed on your host machine:

- **Python 3.8+**
- **FFmpeg** (essential for MP3 conversion)
- **Node.js 18+**

#### 2. Installation

Bash

```bash
# Clone the vision
git clone https://github.com/yourusername/my-melody.git
cd my-melody

# Install Python dependencies
pip install yt-dlp

# Install Frontend dependencies
npm install

```

#### 3. Run the Engine (CLI Mode)

To test the core downloader logic:

Bash

```python
python.exe src/lib/downloader.py

```

#### 4. Launch the Dashboard

Bash

```js
npm run dev

```
